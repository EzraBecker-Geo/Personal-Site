Network Analysis Applied to Subsets of the Lithium Mineral Locality Database

A method of visualizing the relationships of mineral systems

About This Project

Networks display data as nodes and represent relationships between them as edges that connect the nodes. We employ three types of networks: unipartite networks, which contain only one type of node; bipartite networks, which include both primary and secondary nodes; and tripartite networks, which consist of primary, secondary, and tertiary nodes.

Networks were created using Python, leveraging the Pyvis library to generate nodes and edges while simulating forces like repulsion and attraction to dynamically control the network layout.

Themes of Network Analysis

  • The evolution of lithium minerals
  • Lithium minerals and their associated paragenetic modes
  • The co-occurring minerals of lithium-bearing pegmatite localities
  • Locality specific networks

Disparate groups of elements create a sharp divide in the network structure. The left side of the network is predominantly silicates, oxides, and phosphates. The right side of the network consists mostly of sulfides, sulfosalts, and native elements. The division between mineral groups is associated with the two main paragenetic modes of mineral formation at the Tanco mine: PM34 and PM33.
