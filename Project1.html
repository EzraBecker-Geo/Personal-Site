The Lithium Mineral Locality Database (LMLD)

A global repository of hard rock lithium mineral locations

About This Project

The Lithium Mineral Locality Database (LMLD) consolidates data from global sources to provide a comprehensive repository of hard rock lithium mineral occurrences. It includes geographic, mineralogical, paragenetic modes, and mineral age information.

Data was sourced from trusted platforms like Mindat.org and the Mineral Evolution Database. This database aims to support researchers and industry professionals in exploring lithium systems.

Features

  • 33 fields covering a wide range of data categories
  • Detailed geographic distribution of lithium minerals
  • Cross-referenced data with public mineralogy databases
